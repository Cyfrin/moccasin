compile (build)¬∂

Compiles a specific Vyper contract or all vyper contracts in the project.

If no contract or contract path is given, this command will: 1. Find all .vy files in the src/ directory 2. Compile each file using the Vyper compiler 3. Output the compiled artifacts to the out/ directory

Use this command to prepare your contracts for deployment or testing.

-k
Only run tests which match the given substring expression. An expression is a Python evaluable expression where all names are

substring-matched against test names and their parent classes. Example: -k ‚Äòtest_method or test_other‚Äô matches all test functions and classes whose name contains ‚Äòtest_method‚Äô or ‚Äòtest_other‚Äô, while -k ‚Äònot test_method‚Äô matches those that don‚Äôt contain ‚Äòtest_method‚Äô in their names. -k ‚Äònot test_method and not test_other‚Äô will eliminate the matches. Additionally keywords are matched to classes and functions containing extra names in their ‚Äòextra_keyword_matches‚Äô set, as well as functions which have names assigned directly to them. The matching is case-insensitive.

install¬∂

Installs the project‚Äôs dependencies. The first argument is the requirements, given as a pip-compatible strings and/or moccasin github formatted dependencies. - Pip-compatible strings download dependencies as regular python packages from PyPI. - Moccasin github formatted dependencies download dependencies from the Moccasin github repository.

Moccasin github formatted dependencies are formatted as:

GITHUB_ORG/GITHUB_REPO@[@VERSION]

Where: - GITHUB_ORG is the github organization or user that owns the repository. - GITHUB_REPO is the name of the repository. - VERSION is the optional version of the repository to download. If not provided, the latest version is downloaded.

Examples: - pcaversaccio/snekmate@0.1.1 # Moccasin GitHub formatted dependency - snekmate==0.1.1 # Pip-compatible string

mox install [-h] [-d] [-q] [requirements ...]
Positional Arguments¬∂
requirements

Requirements, given as a pip-compatible strings and/or moccasin github formatted dependencies.

Named Arguments¬∂

Default: False
